ARGO

Firmware ESP32-S3 per monitorare un BMS JK (serie PB) su Home Assistant, web e OLED.
Install a firmware for an ESP32-S3 that reads a JK-BMS.
Installazione dal browser / Install from the browser
  1. Usa Chrome, Edge o Opera su desktop (Windows / macOS / Linux). Firefox, Safari e i telefoni non sono supportati.
  2. Collega la scheda ESP32-S3 al PC via USB.
  3. Premi Installa, scegli la porta seriale e attendi.
Browser non supportato — usa Chrome/Edge/Opera desktop. Servito su HTTPS richiesto.
Dopo l'installazione / After install
Al primo avvio non ci sono reti WiFi salvate: la scheda crea una rete ARGO-setup. Collegati con il telefono/PC, apri 192.168.4.1 e scegli la tua rete di casa. Poi ARGO è raggiungibile via web (http://<ip>/ o http://jk-bms-<mac>.local).
Il firmware pubblico non contiene credenziali. MQTT/Home Assistant e le reti WiFi si configurano dopo, dal dispositivo. Codice e documentazione: repo del progetto.
